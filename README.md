# CrackDN

## Overview

CrackDN segmentation is a task-specific package derived from the SematicSeg_Dataset subset of CrackDataset_DL_HY.

- Task: segmentation
- Images: 468
- Annotations: 2404
- Classes: 1 (crack)
- Annotation format: paired PNG masks

This package targets pixelwise pavement crack detection and preserves source publication context.

## Source Dataset

- Name: CrackDataset_DL_HY (SematicSeg_Dataset)
